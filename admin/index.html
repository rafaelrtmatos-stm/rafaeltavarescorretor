Integração GitHub
Configure para publicar alterações direto no site

🐙 Como funciona

Após configurar, o botão 🚀 Publicar no site vai commitar suas alterações diretamente no GitHub. A Vercel detecta automaticamente e publica o site em 1-2 minutos.

🔑 Personal Access Token

📋 Como criar o token

1
Acesse github.com/settings/tokens
2
Clique em Generate new token (classic)
3
Dê um nome, ex: admin-site-rafael
4
Marque a caixa repo (primeira opção)
5
Clique Generate token e copie o token gerado
6
Cole abaixo e clique em Salvar

⚠️ O token fica salvo somente no seu navegador (localStorage). Nunca sai do seu computador.
